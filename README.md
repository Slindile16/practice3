# practice3# 
Python Revision Test

This test practices:

- Conditional statements
- Functions
- Basic loops
- Advanced loops
- Simple algorithms

You must complete the functions in `revision.py` so that all tests in `test_revision.py` pass.

Run the tests with:

python3 -m unittest test_revision.py



---------------------------------

SECTION 1 — BASIC LOOPS

sum_to_n(n)
Return the sum of numbers from 1 to n.

count_even(numbers)
Return the count of even numbers in the list.

print_numbers(n)
Return a list containing numbers from 1 to n.


---------------------------------

SECTION 2 — ADVANCED LOOPS

multiplication_table(n)
Return the first n multiples of n.

Example:
n = 3 → [3,6,9]

nested_count(n)
Use nested loops to count the total number of iterations in an n x n loop.

Example:
n = 3 → 9

pyramid_levels(n)
Return a list showing pyramid levels.

Example:
n = 3 → ["*", "**", "***"]


---------------------------------

SECTION 3 — SIMPLE ALGORITHMS

find_max(numbers)
Return the largest number in the list.

find_min(numbers)
Return the smallest number in the list.

average(numbers)
Return the average value of the list.
